# HTML

## Basic structure
- Doctype
- HTML
- HEAD, title
- BODY

## Text elements
- headings: h1, h2, h3, h4, h5, h6
- paragraphs: p
- comments: "<!-- some comments -->"
- bold text: strong
- italic: em

## Lists
- ordered list: ol > li
- un-ordered list: ul > li

## Images
- img: src, alt, width, height

## Hyperlinks
- a: href, target
- internal, external

## Semantic elements
- nav: navigation
- header
- footer
- article
- section
- aside
- Semantic html helps google understand page for seo.
- Semantic html is important for accessibility.
